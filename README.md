# Корпоративный портал АО «НИИ ЭЛЕКТРОМЕРА»

## О проекте

Внутренний корпоративный сайт для сотрудников института.

## Страницы

- Главная (новости, документы, нормативные акты)
- Бланки документов
- Электронный архив стандартов
- Система менеджмента качества (СМК)
- Информационная безопасность

## Технологии

- HTML5, CSS3, JavaScript (ES6+)
- LocalStorage для хранения данных
- ООП архитектура (модульные менеджеры)

## Установка и запуск

1. Скачайте проект
2. Откройте файл `main.html` в браузере (или используйте Live Server в VS Code)

## Системные требования

- Рекомендуемое разрешение экрана: от 1104px и выше
- Планшеты и телефоны: не рекомендуется

## Административный режим

- Код доступа: `1111`
- Как сменить код: откройте `js/modules/Auth.js`, найдите `accessCode = '1111'`, замените на новый код
- Возможности: добавление, редактирование, удаление контента
